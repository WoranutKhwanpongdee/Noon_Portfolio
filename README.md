# Noon's Developer Portfolio

A professional, cyberpunk-themed developer portfolio.

## Project Structure

- `src/`: Main source code
  - `index.html`: Main entry point
  - `css/`: Stylesheets
  - `js/`: JavaScript logic
  - `assets/`: Images, documents, and other static assets
- `scripts/`: Utility scripts (e.g., `recover.py`)
- `archive/`: Original single-file version

## How to Run

Simply open `src/index.html` in any modern web browser.

## Features

- **Interactive Terminal**: Type `help` to explore commands.
- **Cyberpunk Theme**: Toggle between normal and cyberpunk modes.
- **GitHub Activity Graph**: Real-time contribution data.
- **Bento Grid Layout**: Modern skill display.
- **Holographic Experience Card**: Interactive 3D experience highlight.
